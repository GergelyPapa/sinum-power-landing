## 🎨 Customization


### Change Colors

### Change Content
Edit component files in `src/components/`

### Change Text
All text is in Hungarian - edit components directly

## 🆘 Troubleshooting

### Port 3000 in use?
```bash
lsof -i :3000
kill -9 <PID>
```

### Build fails?
```bash
rm -rf .next node_modules
npm install
npm run build
```

### Environment variables not working?
- Must start with `NEXT_PUBLIC_`
- Restart server after changing
- Check in production platform settings

### Styling not working?
```bash
npm run dev  # Tailwind CSS watches for changes
```
