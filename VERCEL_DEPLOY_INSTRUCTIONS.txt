🚀 3-STEP DEPLOYMENT PROCESS:


STEP 1: PUSH TO GITHUB (5 minutes)
─────────────────────────────────────────────────────────────────────────────

1. Create a new GitHub repository:
   → https://github.com/new
   
   Repository name: sinum-power-landing
   Description: SINUM POWER Pre-Landing Page
   Privacy: Public
   
   Click: "Create repository"

2. Copy the repository URL (HTTPS format):
   https://github.com/YOUR_USERNAME/sinum-power-landing.git

3. Push the code:

   cd /tmp/sinum-power
   
   git remote add origin https://github.com/YOUR_USERNAME/sinum-power-landing.git
   git branch -M main
   git push -u origin main

   (When prompted for password, use your GitHub Personal Access Token)


STEP 2: DEPLOY TO VERCEL (2 minutes)
─────────────────────────────────────────────────────────────────────────────

Option A: Via Vercel Dashboard (Easiest)

1. Go to: https://vercel.com
2. Log in (or create account)
3. Click: "Add New" → "Project"
4. Click: "Continue with GitHub"
5. Select: sinum-power-landing
6. Click: "Import"
7. Click: "Deploy"

Wait 2-3 minutes for build to complete.

Your live URL will be:
→ https://sinum-power-landing.vercel.app


Option B: Via Vercel CLI

   npm i -g vercel
   cd /tmp/sinum-power
   vercel --prod
