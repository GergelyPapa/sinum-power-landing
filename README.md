# SINUM POWER - Pre-Landing Page

Modern, high-converting pre-landing page for SINUM POWER industrial energy optimization solution.

## Features

✅ **Modern SaaS Design** - Stripe/AI tools style aesthetic
✅ **Professional Color Scheme** - Steel blue (#1e3a8a) + white + light gray
✅ **Industrial-Tech Aesthetic** - Modern, clean, professional look
✅ **Hungarian Language** - Formal "magázódás" tone
✅ **Next.js 14** - App Router, SSR ready
✅ **Tailwind CSS** - Fully responsive design
✅ **Lucide React Icons** - Industrial/energy icons
✅ **Production Ready** - Optimized, minified code
✅ **Conversion Optimized** - High-converting funnel design

## Technology Stack

- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS 3.4
- **Icons**: Lucide React
- **Language**: TypeScript
- **Font**: System sans-serif (optimized for performance)

## Installation & Usage

### Prerequisites
- Node.js 18+
- npm or yarn

### Setup

```bash
# Clone or navigate to project directory
cd sinum-power

# Install dependencies
npm install

# Run development server
npm run dev

# Open browser
# Visit: http://localhost:3000
```

### Build for Production

```bash
# Build optimized production bundle
npm run build

# Start production server
npm start
```
